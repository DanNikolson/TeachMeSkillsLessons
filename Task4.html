Знакомство с тегами
Тэг Семантика Блочный/Строчный
<aside></aside> Элемент <aside> (от англ. aside — в стороне, отступление) представляет собой раздел страницы, который имеет косвенное отношение к содержимому страницы и может быть рассмотрен отдельно от этого содержимого. <aside> применяется для боковых панелей, рекламных блоков, ссылок на архив, меток и другой информации, которая отделена от основного содержимого страницы. блочный
<article></article> Элемент <article> (от англ. article — статья) представляет собой независимый фрагмент веб-страницы и, как правило, включает «шапку», основное содержимое и «подвал», в которых располагаются такие части, как заголовок, текст, имя автора, дата публикации, метки, рейтинг статьи и др блочный
<body></body> HTML-элемент <body> представляет собой контент (содержимое) документа HTML. Ни то,ни другое
<button></button> Элемент <button> (от англ. button — кнопка) создаёт на веб-странице кнопки и по своему действию напоминает результат, получаемый с помощью <input> (с атрибутом type="button | reset | submit"). строчно-блочный
<footer></footer> Элемент <footer> (от англ. footer — нижний колонтитул, подвал) задаёт «подвал» сайта или раздела веб-страницы, в нём может располагаться имя автора, дата документа, контактная и правовая информация. блочный
<h1></h1> HTML предлагает шесть заголовков разного уровня, которые показывают относительную важность секции, расположенной после заголовка. Так, элемент <h1> (от англ. heading — заголовок) представляет собой наиболее важный заголовок первого уровня блочный
<head></head> Элемент <head> (от англ. head — голова) предназначен для хранения других элементов, цель которых — помочь браузеру в работе с данными. Ни то,ни другое
<header></header> Элемент <header> (от англ. header — верхний колонтитул, шапка) задаёт «шапку» сайта или раздела веб-страницы. блочный
<html></html> Элемент <html> является контейнером, который заключает в себе всё содержимое веб-страницы, включая элементы <head> и<body> Ни то,ни другое
<img src="" alt=""> Элемент <img> (от англ. image — изображение) предназначен для отображения на веб-странице изображений в графическом формате GIF, JPEG, SVG или PNG. строчный
<li></li> Элемент <li> (от англ. list item — пункт списка) определяет отдельный пункт списка блочный
<link rel="stylesheet" href=""> Устанавливает связь с внешним документом вроде файла со стилями или со шрифтами. Ни то,ни другое
<main></main> Элемент <main> (от англ. main — основной, главный) предназначен для основного содержимого документа. Ни то,ни другое
<meta> Элемент <meta> (от англ. metadata — метаданные) определяет данные (они называются ещё метатеги), которые используются для хранения информации, предназначенной для браузеров и поисковых систем. Ни то,ни другое
<p></p> Определяет текстовый абзац. блочный
<script></script> Элемент <script> (от англ. script — сценарий) предназначен для описания скриптов, может содержать ссылку на программу или её текст на определённом языке. строчный
<section></section> Задаёт раздел документа, может применяться для блока новостей, контактной информации, глав текста, вкладок в диалоговом окне и др. блочный
<span></span> Универсальный строчный элемент <span> (от англ. span — охватывать) предназначен для выделения отдельных строк, символов или других строчных элементов для дальнейшего изменения их оформления с помощью стилей. строчный
<title></title> Определяет заголовок документа. Ни то,ни другое
<ul></ul> Элемент <ul> (от англ. unordered list — неупорядоченный список) устанавливает маркированный список. блочный
